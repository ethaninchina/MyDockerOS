### Easy Mock 是一个可视化，并且能快速生成模拟数据的持久化服务
#### 使用方式
1. 安装 docker-compose
2. 下载 `docker-compose.yml`和`default.json` 2个文件, 创建文件夹 `mkdir -p ./data/{db,redis} logs && chmod 777 ./logs`
3. 启动：`docker-compose up -d`

web访问: http://localhost:7300
